## Project setup

**JSON_SERVER_API**
```
step I: install json server

npm i -g json-server

step II: install nodemodule

npm i
```
**VUE_PROJECT_CRUD_MEDICAL**

### Compiles and hot-reloads for development

json-server datas.json

STEP II: run VUE_PROJECT_CRUD_MEDICAL

npm run serve
